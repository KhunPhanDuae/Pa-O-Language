ပအိုဝ်ႏသဒ္ဒါႏထွို့ꩻဗူႏပွုံႏ (Pa'O Grammar Sentence Structure)

ပအိုဝ်ႏသဒ္ဒါႏကရိုꩻနဝ်ꩻ ပအိုဝ်ႏဘာႏသာႏငဝ်းငွါ ထွို့ꩻဗူႏပွုံႏ၊ ငဝ်းငွါစီꩻကန်တꩻဖွောင်ꩻတွမ်ႏအခင်ႏလမ်းအုံ အသွုံꩻဖုံႏ ဝါကျငဝ်းငွါစုဗူႏဖုံႏနဝ်ꩻသွူ။

ဝါစင်္ဂငဝ်း (၈) ဗာႏ

၁။ နာမ်ႏ (Noun)
၂။ နာမ်ႏတင် (Pronoun)
၃။ ကြိယာႏ (Verb)
၄။ နာမဝိသေသန ( Adjective )
၅။ ကြိယာႏဝိသေသန ( Adverb )
၆။ ငဝ်းသွုပ် ( Conjunction )
၇။ ခေးထွိုင် ( Preposition)
၈။ တဲႏအွဉ် ( Interjection )
၉။ နွို့သက်မွန် လက္ခဏာႏ ( Articles/Determiner )

Pa'O - Parts of Speech

နာမ်ႏ (Noun)

  1. လိုꩻ (human)
  2. ခွန်တဲးဝါး (person)
  3. နင်ႏဖုဲင်ႏရက် (person)
  4. ကွန်ပျူတာ (computer)
  5. လမ်း (house)
  6. ဒုံ (village)
  7. ကျောင်ꩻ (school)
  8. ထီ (water)

ကြိယာႏ (Verb)

  1. အံႏ (eat)
  2. လွေꩻ (go)
  3. အုံ (live)
  4. သဲင်ꩻ (return)
  5. ထွိုင်ႏ (get up)
  6. ဗေင်ႏ (sleep)
  7. ငွိုး (cry)
  8. င (laugh)

နာမဝိသေသန (Adjective)

  1. တန် (big)
  2. ပေႏ (small)
  3. တရေꩻ (beautiful)
  4. ဟွမ် (မွှေး)
  5. အရောင်ႏ (color)

ကြိယာႏဝိသေသန (Adverb)

  1. မွိုင်မွိုင် (မြန်မြန်)
  2. မွူးရဝ်ꩻခိုႏ (morning)
  3. ယွယွ (slowly)
  4. ဆေ့ꩻဆေ့ꩻ (little)

နာမ်ႏခြောင်ꩻတင် (Pronoun)

  1. ခွေသွဉ်း (i'm)
  2. ဝွေꩻ (he)
  3. ခွေ (i)
  4. နာꩻ (you)
  5. နီ (we)
  6. သီ (မင်းတို့)

ငဝ်းသွုပ်ဆွိုက် (Conjunction)

  1. တွမ်ႏ (and)
  2. ကွဲးတွော့ꩻ (but)
  3. မွေးတဝ်းလဲ့ (or)

ခေးထွိုင် / အခိန်ႏခင်ႏလမ်း (Preposition)

  1. အဝ်ႏမုဲင်ꩻ (ဘယ်မှာ)
  2. အခိန်ႏမုဲင်ꩻ (ဘယ်အချိန်)
  3. အဝ်ႏကို (ဘယ်ထဲ့)
  4. အွိုးမုဲင်ꩻ (ဘယ်နား)

ငဝ်းတဲႏအွဉ် (Interjection)

  1. အံႏဖြာႏ (wow!)
  2. အိုး!
  3. ဝိုးဟိုး!
  4. ဟွေး၊ အွိုး၊ အွူ၊ ဟွူႏ၊ အွောန်ႏ

လိတ်သွဉ်ခန်ꩻအကွို့ꩻ(Lesson) - ၁

ငဝ်းငွါခြွဉ်းဗူႏပွုံႏ (Sentence Structure):

ပအိုဝ်ႏငဝ်းနဝ်ꩻ ကုဲင်းတဲမ်းဒါႏနေးတွမ်ႏ S + V + O (Subject + Verb + Object) ထွို့ꩻဗူႏပွုံႏ နဝ်ꩻသွူ။
Example structure: Subject + Verb + Object

၂။ ပြဲင်ႏခေါ် ငဝ်းဖြုံႏဖုံႏ (Word Comparison Table)

Function ပအိုဝ်ႏ (Pa'O) မြန်မာ(Burmese) English

ဥပမာႏ: ခွေ + လွေꩻ + ကျောင်ꩻ။

၃။ သွုံꩻလွေꩻဥပမာႏ (Usage Example)

ပအိုဝ်ႏ: ခွေလွေꩻကျောင်ꩻ။

မြန်မာ: ကျွန်တော် ကျောင်းသို့သွားသည်။

English: I go to school.

သတိပြုရန်: ပအိုဝ်းစကားသည် မြန်မာစကားအတိုင်း V-final structure မဟုတ်ဘဲ SVO သဒ္ဒါစဉ်ကို အသုံးပြုသော ဘာသာဖြစ်သည်။ ပညာရေးဆိုင်ရာသင်ကြားရေးအတွက် လွယ်ကူအောင် ပုံစံချဖော်ပြရန် အရေးကြီးသည်။
